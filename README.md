## Easy Haskell Development Setup with Docker

```bash
$ docker run -it --rm -v $(pwd):/source haskell:8.0
GHCi, version 8.0.1: http://www.haskell.org/ghc/  :? for help
Prelude> :cd source/
Prelude> :l Core
```
